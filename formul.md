## **Вывод закона Ома в дифференциальной форме**

### **1. На макроуровне (из классического закона Ома)**

| Шаг | Формула | Комментарий |
|-----|---------|-------------|
| **1.** Закон Ома для участка цепи | $ I = \frac{U}{R} $ | Исходная формула |
| **2.** Подстановка сопротивления проводника | $ jS = \frac{E\,l}{\rho\,l/S} $ | $U = E l$ (однородное поле), <br> $R = \rho \frac{l}{S}$ |
| **3.** Преобразование | $ jS = \frac{E l S}{\rho l} $ | Упрощение дроби |
| **4.** Сокращение | $ j = \frac{E}{\rho} $ | Сокращаем $S$ и $l$ |
| **5.** Введение проводимости | $ j = \sigma E $ | $\sigma = \dfrac{1}{\rho}$ — удельная проводимость |

**Результат:**  
$ \boxed{j = \sigma E} $
— **закон Ома в дифференциальной форме**.

---

### **2. На микроуровне (классическая электронная теория)**

| Шаг | Формула | Комментарий |
|-----|---------|-------------|
| **1.** Определение силы тока | $ I = \frac{q}{t} $ | Исходное определение |
| **2.** Через количество носителей | $ I = \frac{N e}{t} $ | $q = Ne$, где $e$ — заряд электрона |
| **3.** Через концентрацию | $ I = \frac{n V e}{t} $ | $N = nV$, $n$ — концентрация |
| **4.** Выражаем плотность тока | $ jS = \frac{n S l e}{t} $ | $V = Sl$ (объём цилиндра) |
| **5.** Упрощаем | $ j = \frac{n l e}{t} $ | Сокращаем $S$ |
| **6.** Вводим скорость дрейфа | $ j = n e u $ | $u = \dfrac{l}{t}$ — средняя скорость дрейфа |

**Промежуточный результат:**  
$ \boxed{j = n e u} $
— связь плотности тока со скоростью дрейфа.

---

### **3. Связь скорости дрейфа с напряжённостью поля**

| Шаг | Формула | Комментарий |
|-----|---------|-------------|
| **1.** Второй закон Ньютона | $ F = m a $ | |
| **2.** Сила, действующая на электрон | $ E e = m a $ | $F = E e$ |
| **3.** Ускорение электрона | $ a = \frac{E e}{m} $ | |
| **4.** Скорость дрейфа | $ v = \frac{E e \tau}{m} $ | $v = a\tau$, где $\tau$ — среднее время свободного пробега |
| **5.** Подстановка в $j = n e u$ | $ j = n e \cdot \frac{E e \tau}{m} $ | |
| **6.** Преобразование | $ j = \frac{n e^2 \tau}{m} \, E $ | |
| **7.** Идентификация проводимости | $ \sigma = \frac{n e^2 \tau}{m} $ | Сравниваем с $j = \sigma E$ |

**Итоговый результат:**  
$
\boxed{\sigma = \frac{n e^2 \tau}{m}}
\qquad
\boxed{j = \sigma E}
$

---

## **Ключевые выводы**

1. **Закон Ома в дифференциальной форме:**  
   $ j = \sigma E $

2. **Плотность тока через скорость дрейфа:**  
   $ j = n e u, \quad \text{где } u \text{ — скорость дрейфа} $

3. **Удельная проводимость металлов (формула Друде–Лоренца):**  
   $ \sigma = \frac{n e^2 \tau}{m} $

4. **Скорость дрейфа в поле $E$:**  
   $ u = \frac{e \tau}{m} E $
   — пропорциональна напряжённости поля.

Формулы демонстрируют связь макроскопических ($I, U, R$) и микроскопических ($n, e, \tau, m$) параметров в классической теории проводимости.
